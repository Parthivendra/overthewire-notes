# OverTheWire — Bandit

> A personal knowledge base documenting my journey through the OverTheWire Bandit wargame.

---

## Overview

Bandit is a Linux-focused wargame designed to teach command-line fundamentals, file operations, permissions, text processing, networking concepts, and problem-solving through progressively challenging levels.

The objective is to obtain the password for the next level by analyzing the current environment and applying Linux commands effectively.

## Level Note Template

Each level note should contain:

### Objective

What the level requires.

### Enumeration

Observations made before attempting the solution.

### Commands Used

Commands executed and their purpose.

### Solution

The exact process used to obtain the password.

### Lessons Learned

Key concepts learned from the level.

### References

Relevant man pages, documentation, or notes.
